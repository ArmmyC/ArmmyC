## About me

I'm a Computer Engineering student at **King Mongkut's University of Technology Thonburi (KMUTT)** who enjoys turning ideas into working prototypes. My work usually sits between hardware and software: processors, embedded systems, robotics, AI-powered tools, data systems, and web apps.

I like projects that are practical, testable, and easy for other people to understand.

```text
Current interests
├── Digital design: Verilog, RISC-V, SAP-1, CPU datapaths
├── Embedded systems: ESP32, Arduino, sensors, robotics, IoT
├── AI systems: LLM hosting, SQL/RAG agents, local-first AI apps
├── Software tools: FastAPI, React, Python, databases, automation
└── Learning style: build, document, test, improve
```

- **Build things that teach something.**
